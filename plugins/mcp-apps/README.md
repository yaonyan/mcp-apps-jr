# MCP Apps Plugin for Claude Code

A Claude Code plugin that provides skills for building MCP Apps.

## Installation

Install via Claude Code:

```
/plugin marketplace add modelcontextprotocol/ext-apps
/plugin install mcp-apps@modelcontextprotocol-ext-apps
```

## Usage

Invoke skills by asking Claude Code to:

### Create MCP App

- "Create an MCP App"
- "Add a UI to an MCP tool"
- "Build an interactive MCP View"
- "Scaffold an MCP App"

### Migrate from OpenAI

- "Migrate from OpenAI Apps SDK"
- "Convert OpenAI App to MCP"
- "Port from window.openai"
